# Backend Railway

Backend Spring Boot listo para Railway.

En Railway:

- Root Directory: vacío, siempre que `pom.xml` y `src/` estén en la raíz del repositorio backend.

Variables CeroPapel recomendadas:

```text
CERO_PAPEL_BASE_URL=https://ceropapel-integracion-api.dssm.cl
CERO_PAPEL_AUTH_PATH=/auth/login
CERO_PAPEL_PURCHASE_REQUESTS_PATH=/purchase-requests/abastecimiento
CERO_PAPEL_RUT=15582517-0
CERO_PAPEL_PASSWORD=********
```
